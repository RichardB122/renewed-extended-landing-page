# LOTR Renewed Extended Landing Page Project
This is a simple landing page for the LOTR: Renewed Extended mod. It uses Astro, React and Tailwind (v4) and is written primarily in TypeScript based on an initial raw HTML/Tailwind mockup made by Claude.

The mockup was created by Claude and is available in mockup.html

## Installing / Running

#### Install
```npm install
```

#### Running
```npm run dev
```

#### Build
Builds the static site
```npm run build
```
